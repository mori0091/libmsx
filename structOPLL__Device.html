MSX Function pointer to write data to a OPLL register.

Parameters

  • 0x0e : RHYTHM control register
  • 0x0f : TEST register
  • 0x10..0x18 : F-Number (LSB 8 bits) registers
  • 0x20..0x28 : SUS/KEY/BLOCK/F-Number (MSB) registers
