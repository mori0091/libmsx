Executes VDP command "LMMM" (copy rectangular area w/ logical operation).

This function is equivalent to the following code:

struct vdp_cmd c;
vdp_cmd_set_SX(&c, x);
vdp_cmd_set_SY(&c, y);
vdp_cmd_set_DX(&c, x2);
vdp_cmd_set_DY(&c, y2);
vdp_cmd_set_NX(&c, w);
vdp_cmd_set_NY(&c, h);
vdp_cmd_set_ARG(&c, 0);
vdp_cmd_set_logop(&c, logop);
vdp_cmd_execute(&c, VDP_CMD_LMMM);
